1. O čo presne ide

Jedna veta: Pred platbou alebo akciou zavoláš GET /api/v1/agents/{id}/status. Agent sa registruje sám (Lightning + Ed25519). Pri väčších sumách: ?include=cert_proof.

Problém

  • Ktokoľvek môže spustiť tisíce „agentov“ bez identity.
  • API kľúče a SaaS registrácie nie sú určené pre stroje.
  • Platformy nevedia, komu dovoliť platbu alebo akciu.

Čo ponúkaš

  • M2M registrácia: POST /api/v1/register, platba Lightning, certifikát.
  • Gate: GET /api/v1/agents/{id}/status → verified.
  • Plug-in: /integrators, umb_live_ len rate limit (nie identita agenta).
  • Nekustodiálne: hub nedrží prostriedky agentov.
