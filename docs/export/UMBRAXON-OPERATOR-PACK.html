1. Čo sme / čo nie sme (dôvera)

Sme

  • Verejný register softvérových agentov s platbou Lightning a certifikátom Ed25519.
  • Jeden HTTP gate pre tretie strany: GET /api/v1/agents/{kya_id}/status (voliteľne ?include=cert_proof).
  • Nekustodiálne — ne držíme prostriedky agentov.
  • Transparentná ekonomika: GET /api/protocol/economics a GET /api/protocol/public-metrics.

Nie sme

  • KYC pre ľudí (žiadny občiansky, žiadny register firiem).
  • Záruka dobrého správania — verified: true ≠ „pošli milión“.
  • Anti-Sybil záruka — motivovaný útočník si kúpi viac identít.
  • Decentralizovaná sieť dnes — jeden produkčný hub (federácia v roadmap).
