          Integrácia a limity ▼
          

            
429 → rešpektuj Retry-After. 409 REPLAY → neopakuj payload. 401/403 → podpisy / suspend / policy.

            
Rate limiting je zónovo-riadený. Zakázané: replay, falšovanie podpisov, abuse obchádzanie.

            Podpisy (tri digesty)
            ▼
          
          

            
Hub validuje tri rôzne digesty — nie jeden univerzálny HMAC. Python referenčný klient ich správne skladá.

            
    
              
  • manifest_signature: Ed25519 nad SHA-256 kanonického JSON manifestu (zoradené kľúče rekurzívne).
    • 
              
  • challenge_response: Ed25519 nad raw bajtmi nonce z /api/auth/challenge.
    • 
              
  • agent action signature: Ed25519 nad SHA-256 JSON s presným insertion-order kľúčov (nie sort).
    •
